## Installation

```sh
# Install dependencies in Nix environment
$ nix develop
# or Use direnv
$ direnv allow

# or Install dependencies
$ pnpm install

# Launch dev server
$ just dev

# Build
$ just build

# Preview build artifacts
$ just preview

# Run static checks
$ just check

# Deploy to CloudFlare pages by wrangler
$ just deploy
```
