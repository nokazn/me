## Installation

```sh
# Install dependencies in Nix environment
$ nix develop
# or Use direnv
$ direnv allow

# or Install dependencies
$ pnpm install

# Launch dev server
$ pnpm run dev

# Build
$ pnpm run build

# Preview build artifacts
$ pnpm run preview

# Run static checks
$ pnpm run check

# Deploy to CloudFlare pages by wrangler
$ pnpm deploy
```
